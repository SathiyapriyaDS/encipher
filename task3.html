Image Formats
JPEG
JPEG is a type of picture file. It makes photos smaller in size, so they are easy to save and share. The file name usually ends with .jpg or .jpeg.
PNG
PNG stands for Portable Network Graphics. It is a type of image file that keeps high quality and supports transparent backgrounds. The file name ends with .png.
SVG
SVG stands for Scalable Vector Graphics. It is a type of image file used for drawings, icons, and logos
